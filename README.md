# climate-hustlers-mbale
Stata analysis of gendered livelihood pathways among climate-induced migrant youth in Mbale, Uganda.
## Project Overview

This project analyzes gendered livelihoods(hustles) among climate-induced migrant youth in Mbale, Uganda.

The study explores:
Gender differences in unemployment
Informal hustle typologies
Asset accumulation patterns
Agency under constraint
Capability and mental wellbeing outcomes

The analysis is conducted in Stata using logistic, multinomial, and OLS regression models.

Methods
Logistic regression (unemployment, agency)
Multinomial logistic regression (hustle typology)
OLS regression (capability, PHQ-9 score)
Index construction (agency, assets, capability)
